## 技术栈

**:mega: 列了一大堆，搞得好像都用的很好似的 :sweat_smile::sweat_smile:**

微信小程序：**uni-app**

pc前端：**vue2 + vuex + vue-router + vue-cli3 + element-ui + ES6**

后端：**node + egg.js + ES6**
